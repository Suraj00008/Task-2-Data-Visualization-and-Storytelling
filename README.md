# Task 2 - Data Visualization and Storytelling

## Objective
Create visualizations that convey business insights using Tableau.

## Dataset
Sample Superstore Dataset

## Visualizations Created
1. Sales Trend Analysis
2. Sales by Category
3. Profit by Region
4. Sales by Segment
5. Top 10 Products by Sales

## Key Insights
- Consumer segment contributes the highest sales.
- Technology category generates the highest revenue.
- Sales peak during November and December.
- West region generates the highest profit.
- Top products contribute significantly to revenue.

## Tool Used
- Tableau Public
